# 数据库设计

* 用户

| 字段名   | 含义              | 类型    |
| -------- | ----------------- | ------- |
| id       | id                | key     |
| password | 密码              | varchar |
| account  | 账号 使用邮箱注册 | varchar |
| avatar   | 头像编号          | int     |
| name     | 昵称              | varchar |

* 备忘录

| 字段名   | 含义               | 类型    |
| -------- | ------------------ | ------- |
| id       | id                 | key     |
| info     | 备忘录中存储的信息 | varchar |
| owner    | 拥有者             | bigint  |
| status   | 状态               | int     |
| end_time | 截止时间           | bigint  |

* 密码

| 字段名 | 含义             | 类型   |
| ------ | ---------------- | ------ |
| id     | id               | key    |
| use    | 这段密码由于什么 | string |
| owner  | 密码所属用户账号 | string |
| info   | 加密后的密码     | text   |






# 接口开发进度

| 接口路径        | 请求方式 | 功能                 |
| --------------- | -------- | -------------------- |
| /api/user/mail  | POST     | 请求发送邮箱验证码   |
| /api/user       | POST     | 注册用户             |
| /api/user/login | POST     | 登录                 |
| /api/memo       | GET      | 根据拥有者查询备忘录 |
| /api/memo       | DELETE   | 删除备忘录           |
| /api/memo       | POST     | 上传备忘录           |
| /api/memo       | PUT      | 更新备忘录           |
| /api/cipher     | POST     | 保存密码             |
